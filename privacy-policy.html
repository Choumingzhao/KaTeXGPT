Information Collection and Use

KaTeXGPT - ChatGPT to Word Equation Converter extension does not collect, store, or transmit any personal information. The only data stored is:

  • Installation date (stored locally)
  • Number of equations converted (stored locally)
  • Achievement milestones reached (stored locally)

Local Storage

All data is stored locally on your device using browser's local storage and is never transmitted to external servers.

Permissions

The extension requires the following permissions:

  • activeTab: To interact with equations on chat.openai.com
  • clipboardWrite: To copy converted equations
  • scripting: To inject necessary scripts for equation conversion
